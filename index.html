FAQ

Questions fréquentes

Est-ce vraiment gratuit ? →
Oui, 100%. Le hub tourne sur GitHub Pages (gratuit), tes données restent dans ton navigateur et ton backup dans ton repo GitHub privé (gratuit). Aucun serveur, aucun abonnement.
Behance demande-t-il une clé API ? →
Non. Adobe a fermé l'API Behance, alors la sync utilise le flux RSS public de ton profil : ton pseudo suffit. Chaque projet arrive avec son lien et sa couverture.
Comment mes clients laissent-ils un avis ? →
Sur ton site publié, bouton « ✎ Laisser un avis » : nom, étoiles, texte. L'avis arrive dans ton dashboard où tu l'approuves ou le refuses. Les avis approuvés s'affichent sur ton site au prochain déploiement.
Où sont stockées mes données ? →
Dans ton navigateur (localStorage) et sauvegardées automatiquement dans un repo GitHub privé qui t'appartient. Tu peux tout exporter en JSON à tout moment.
Puis-je utiliser mon propre domaine ? →
Ton site est publié sur username.github.io/ton-repo. GitHub Pages supporte les domaines personnalisés — la configuration guidée depuis le hub arrive dans la roadmap.
