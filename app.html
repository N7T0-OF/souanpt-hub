Sauvegarde complète du Hub
Exporte toutes tes données (projets, clients, factures, avis, portails, paramètres) dans un seul fichier. C'est ta sauvegarde — et le point de départ de la future migration vers le cloud (Firebase). Le token GitHub n'est jamais inclus.
💡 Tes données V1 ne sont pas perdues : elles sont dans ta sauvegarde GitHub privée (ton-compte-hub-data). Connecte GitHub (token) puis clique « Restaurer » — projets, clients, factures et avis reviennent, et se synchronisent ensuite dans le cloud automatiquement.
